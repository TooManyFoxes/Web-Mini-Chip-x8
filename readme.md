## Web Mini Chip Emulator

This program emulates the capabilities of the WMC, written
in MistWarp.

You can view the docs in the "docs.md" file.

To write programs for the WMC, download [this](https://marketplace.visualstudio.com/items?itemName=roxy.webminichip-mcasm-x8&ssr=false#overview) extension for Visual Studio Code!

To write a WMC program, end your file with .mcasm. Then to compile, CMD / CTRL + SHIFT + P, then type mcasmcompile!
